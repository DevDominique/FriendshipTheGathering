# web103_unit3_project# WEB103 Project 3 - *Friendship: The Gathering *🧙🏾‍♀️

Submitted by: **Dominique Wilson**

About this web app: ***Friendship: The Gathering* is a community web app designed for Dentonites who want to make new friends through fun, in-person events. Users browse events happening around town and explore locations that host them. Built with React, Express, Node.js, PostgreSQL, and Bootstrap — the project focuses on clean routing, dynamic data rendering, and a fully connected backend. Its main objectives are to practice full‑stack development, build a functional events directory, and create an inviting experience that reflects the spirit of real‑world community building. This project showcases responsive UI, RESTful API design, and database‑driven content.**

Time spent: **24** hours

## Challenges & Lessons

This project pushed me more than the previous week’s assignment, largely because I struggled with the starter code. It took time to understand someone else’s structure, intentions, and patterns, and I realized that this is a core part of real‑world development. In practice, I’ll constantly be reading, debugging, and refactoring code I didn’t write, and the goal is to stay focused on the project’s objectives rather than my personal coding style.

I also ran into a difficult SSL configuration issue that cost me hours, but working through it taught me how to troubleshoot environment variables, database connections, and backend errors more systematically. Overall, this project strengthened my ability to navigate unfamiliar codebases, solve problems under pressure, and build cleaner, more efficient solutions.
